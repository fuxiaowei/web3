# 第1期课程总览

## 🎯 课程目标

通过本课程的学习，掌握Golang的基础语法和进阶特性，为后续的区块链开发打下坚实基础。

## 📊 课程概览

| 部分 | 内容 | 时长 | 难度 |
|------|------|------|------|
| 基础语法 | 环境搭建、数据类型、控制流、函数、结构体 | 40分钟 | ⭐ |
| 进阶特性 | 接口、并发编程、并发安全、标准库 | 80分钟 | ⭐⭐ |
| 实战项目 | Web爬虫综合项目 | 实践 | ⭐⭐⭐ |

## 📚 课件资源

### 基础语法课件 (`courseware/basics.md`)

**第一部分：环境搭建 (5分钟)**
- Go安装与配置
- GOPATH vs GOPROXY
- VS Code配置

**第二部分：基础数据类型 (10分钟)**
- 基本类型与复合类型
- 切片和映射的使用
- 指针基础

**第三部分：控制流 (10分钟)**
- if/else、switch、for循环
- defer语句
- panic和recover

**第四部分：函数和结构体 (10分钟)**
- 函数定义与多返回值
- 闭包的使用
- 结构体和方法

**第五部分：实践练习 (5分钟)**
- 简单计算器程序

### 进阶特性课件 (`courseware/advanced.md`)

**第一部分：接口和多态 (15分钟)**
- 接口定义和实现
- 空接口和类型断言
- 接口组合
- 实战：多态性系统

**第二部分：Goroutine入门 (15分钟)**
- Goroutine创建和使用
- Channel通信机制
- Select语句
- 实战：生产者消费者模式

**第三部分：并发安全 (15分钟)**
- Mutex和RWMutex
- WaitGroup同步
- Context上下文控制
- 实战：并发安全的计数器

**第四部分：并发编程最佳实践 (5分钟)**
- 常见错误和陷阱
- 性能优化建议

**第五部分：标准库与项目实战 (15分钟)**
- 常用标准库
- JSON编解码
- 文件操作
- 时间处理
- 加密哈希
- 实战：Web爬虫项目

## 💻 代码示例

### 基础语法示例 (5个文件)

1. **01-types.go** - 数据类型
   - 演示所有Go语言的类型
   - 包含数组、切片、映射、指针的使用

2. **02-control-flow.go** - 控制流
   - if/else、switch、for的使用
   - defer、panic、recover的用法

3. **03-function.go** - 函数
   - 基本函数、多返回值
   - 闭包和高阶函数

4. **04-struct.go** - 结构体
   - 结构体定义和初始化
   - 方法定义和使用

5. **calculator.go** - 计算器实践
   - 完整的计算器实现
   - 包含加减乘除和统计功能

### 进阶特性示例 (4个文件)

1. **01-interface.go** - 接口和多态
   - Shape接口的多实现
   - 空接口和类型断言
   - 接口组合

2. **02-goroutine.go** - 并发编程
   - goroutine的创建
   - channel的通信
   - select语句的使用

3. **03-concurrency-safe.go** - 并发安全
   - Mutex、RWMutex的使用
   - Context控制
   - 并发安全的计数器

4. **04-producer-consumer.go** - 生产者消费者
   - 经典并发模式
   - Worker Pool实现

### 实战项目 (1个文件)

1. **web-crawler.go** - Web爬虫
   - 并发抓取多个URL
   - Worker Pool模式
   - 完整的错误处理
