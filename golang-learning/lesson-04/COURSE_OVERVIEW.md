# 第4期课程总览：区块链基础与 Go-Eth Client

## 🎯 课程目标

通过本课程的学习，你将：

- 从工程视角理解以太坊的**账户模型、交易、区块结构与 Gas 机制**
- 熟悉主网、测试网、本地私有链等**多种网络类型与节点形态**
- 掌握使用 Go-Ethereum (`go-ethereum`) **客户端库连接节点并读取链上数据**
- 能够**查询区块、交易、账户余额**，并实现**事件订阅与断线重连**
- 学会通过 ABI 与 Go 代码进行**智能合约方法调用、交易发送与事件解析**
- 独立实现一个**迷你区块浏览器 / 链上监听服务**作为综合实战项目

## 📚 课件资源

### Ethereum 与 Go-Eth 基础课件 (`courseware/basics.md`)

- 以太坊账户模型：EOA vs 合约账户  
- 交易与区块结构拆解（字段级别说明）  
- Gas、手续费与 EIP-1559 费用模型  
- 主网 / 测试网 / 本地链对比与节点类型  
- 安装 `go-ethereum`，使用 `ethclient.Client` 连接节点  
- 基础操作：获取链 ID、当前区块高度、简单健康检查  
- 简单连接池与多节点读写策略（示例：`examples/10-multi-node-pool/main.go`）  

### 事件订阅与合约交互课件 (`courseware/advanced.md`)

- 区块与交易的查询模式与性能考量  
- 事件订阅接口（新区块、pending 交易、日志）与通道模型  
- 断线重连与订阅重建的稳健实现思路  
- ABI 编解码：加载 ABI 文件、构造调用数据、解析返回值  
- 使用绑定代码或通用接口调用合约方法、发送交易  
- 解析合约事件日志（topics + data），特别是 ERC-20 事件  

### 项目实战课件 (`courseware/project.md`)

- 迷你区块浏览器需求分析与接口设计  
- 区块 / 交易 / 地址 / 事件等基础查询能力  
- 使用 Go-Eth Client 构建后端服务：模块划分与依赖管理  
- 实时监听 ERC-20 转账并记录到本地存储（内存 / 文件 / 数据库）  
- 对外暴露 HTTP API 或 CLI 命令行接口  
- 部署与运维要点（节点选择、RPC 限流、错误处理与重试）  

## 💻 代码示例

### 基础示例 (`examples/basics`)

1. **01-connect-node.go** - 连接本地/远程以太坊节点，获取链 ID 与最新区块号  
2. **02-block-ops.go** - 获取最新区块、指定区块及区块区间，并解析区块信息  
3. **03-tx-ops.go** - 查询交易详情与回执、交易状态与 Gas 消耗  
4. **04-account-balance.go** - 查询账户 ETH 余额与代币余额（ERC-20）  

### 进阶示例 (`examples/advanced`)

1. **01-subscribe-blocks.go** - 订阅新区块头，实时打印高度与哈希  
2. **02-subscribe-pending.go** - 订阅 pending 交易流，做简单过滤与计数  
3. **03-subscribe-logs.go** - 基于日志过滤器订阅指定合约事件  
4. **04-reconnect-strategy.go** - 订阅断线后的重连与重建订阅逻辑示例  
5. **05-contract-interact.go** - 通用 ABI 方式调用合约方法、发送交易并解析事件  

### 项目示例 (`examples/project`)

1. **block_explorer/** - 迷你区块浏览器服务  
   - 支持按区块号/哈希查询区块  
   - 支持按哈希查询交易与回执  
   - 支持按地址查看最近交易与余额  
   - 支持查看最近监听到的 ERC-20 转账事件  

> 具体文件和实现可在后续迭代中逐步补全，上述为推荐的示例划分与命名。
